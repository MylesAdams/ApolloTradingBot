# INSTRUCTIONS TO BUILD PROJECT

1. Install CMake
    * macOS
        * brew install cmake
    * Ubuntu
        * sudo apt install cmake
    * Windows
        * [Link to CMake](https://cmake.org/download/)
2. Install curl
    * macOS
        * brew install curl
    * Ubuntu
        * sudo apt install curl
    * Windows
        * sudo apt install curl
3. Run commands below with in ApolloTradingBot directory
    * cd Build
    * cmake ../
    * make
